Function for scanning the environment by rotating the robot in place.

This function rotates the robot in place by publishing a Twist message with a positive angular.z value to the ‘cmd_vel’ topic. It does this for 10 iterations at a rate of 10Hz, and then sleeps for 5 seconds.

scripts.finite_state_machine.set_map_coordinates()[source]

Initially, a list of room names is generated. For each room in the list, the ArmorClient is used to query and retrieve the respective X and Y coordinates. These values are stored in variables ‘X’ and ‘Y’ from the map after it’s loaded, respectively. Subsequently, the function populates the ‘coordinates’dictionary with the room name as the key and a nested dictionary containing the X and Y coordinates as the value.
